AI-Driven Smart SLA Monitoring System
An automated, real-time performance evaluation platform built using the MERN Stack. This system helps organizations monitor Service Level Agreements (SLA) for support tickets, providing proactive alerts and real-time tracking to eliminate service delays.

Key Features
Real-Time Dashboards: Live countdown timers for active tickets using Socket.io.

Smart Alerting System: Automated visual and auditory notifications at 80% SLA threshold and 100% breach.

Priority Management: Dynamic deadline calculation based on priority levels (P1, P2, P3).

Performance Analytics: Comprehensive reports for Admin to track Resolved vs. Breached tickets.

Proactive Monitoring: Background status checks powered by Node-Cron.

🛠️ Tech Stack
Frontend: React.js, Tailwind CSS

Backend: Node.js, Express.js

Database: MongoDB Atlas (NoSQL)

Real-time Engine: Socket.io

Automation: Node-Cron
